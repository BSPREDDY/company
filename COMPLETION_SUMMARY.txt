================================================================================
                        🚀 QUICK START (5 Minutes)
================================================================================

1. Start MongoDB:
   mongod

2. Start Server (Terminal 1):
   cd server
   npm install
   npm run dev

3. Start Admin (Terminal 2):
   cd admin
   npm install
   npm run dev

4. Start Client (Terminal 3):
   cd client
   npm install
   npm run dev

5. Create Admin Account:
   - Go to http://localhost:5174/register
   - Fill in email, password, name
   - Click Register

6. Access Applications:
   - Admin: http://localhost:5174
   - Client: http://localhost:5173
   - Server: http://localhost:5000

                      📞 SUPPORT & TROUBLESHOOTING

If you encounter issues:

1. First: Check TROUBLESHOOTING.md (covers 80% of issues)
2. Look for [v0] logs in browser console (F12)
3. Check server terminal for errors
4. Verify: mongod running, .env configured, ports correct

Most Common Issues:
  - "Connection refused" → Server not running (start it!)
  - "Invalid password" → Create account first via register
  - "CORS error" → Restart server
  - "MongoDB error" → Start mongod

All covered in detail in TROUBLESHOOTING.md
